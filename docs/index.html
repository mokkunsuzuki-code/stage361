Stage361: Revocation Proof Injection Gate

Stage361 adds a revocation proof injection gate on top of Stage360 External Timestamp Binding.

Core Meaning

Stage360 asks: when did this evidence exist?

Stage361 asks: is the related key or certificate revoked?

Decision

pending_revocation_proof

This is the correct default state because Stage361 does not yet claim real OCSP or CRL cryptographic verification.

Safety Boundary

  • No private keys
  • No raw secrets
  • No raw QKD key material
  • No false OCSP verified claim
  • No false CRL verified claim
